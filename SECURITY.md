# Security Policy

We take security seriously. If you discover a vulnerability, please report it responsibly.

## Reporting

Email: security@example.org

Include:

- Description of the issue and impact
- Reproduction steps or proof-of-concept
- Affected files, endpoints, or configuration

## Scope

This repo includes client-side code and infrastructure configuration (Firebase + Supabase). Please avoid disclosing sensitive data publicly.

## Response

We will acknowledge reports and work on a fix as quickly as possible. Thank you for helping keep this project safe.
